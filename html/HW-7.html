Домашнее задание №7

1. Привязать добавление товара в корзину к реальному API.

2. Добавить API для удаления товара из корзины.

3.* Добавить файл stats.json, в котором будет храниться статистика действий пользователя с корзиной. В файле должны быть поля с названием действия (добавлено/удалено), названием товара, с которым производилось действие и временем, когда оно было совершено.

Решение

1. Привязать добавление товара в корзину к реальному API.

Пояснения:

Файлы с кодом находятся в папках корневого каталога:
- "source/project_express_single+DEL/public/js/CartComp.js"
- "source/project_express_single+DEL/server/server.js"

2. Добавить API для удаления товара из корзины.

Пояснения:

Порт был изменен на "localhost:3333".

При создании метода "app.delete" были выполнены следующие изменения кода:

В файле "server.js":
1. Создаем копию метода "app.put" с изменением метода на "app.delete".
2. В методе "app.delete" вместо строки "find.quantity += req.body.quantity;" вписываем строку "find.el += req.el;".

В файле "CartComp.js":
1. Изменяем в методе "remove(item)" строку:
- старая: this.$parent.getJson(`${API}/deleteFromBasket.json`)
- новая: this.$parent.deleteJson(`/api/cart/${item.id_product}`)
2. Удаляем в методе "remove(item)" строку
- if(data.result === 1)

3.* Добавить файл stats.json, в котором будет храниться статистика действий пользователя с корзиной. В файле должны быть поля с названием действия (добавлено/удалено), названием товара, с которым производилось действие и временем, когда оно было совершено.
